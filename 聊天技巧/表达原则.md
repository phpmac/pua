# 表达原则

## 核心原则

**不要说太满、不要太直接** - 保持暧昧和神秘感

---

## 关键原则

### 1. 关系没到那一步，话不能说太满

| 关系进度 | 能说什么 | 不能说什么 |
|---|---|---|
| 刚认识 | 基本信息交流 | 表白、承诺 |
| 聊得好 | "你挺好的" | "我爱你" |
| 睡过之后 | 慢慢表达情感 | 过度承诺 |
| 确定关系后 | 正常表达 | 无 |

### 2. 喜欢一个人，不要直接说

**错误示范：**
- "我喜欢你"
- "我好爱你"
- "你是我喜欢的类型"

**正确表达：**
- "你挺好的"
- "我觉得你也挺不错的"
- "和你聊天挺开心的"
- "和你在一起很舒服"

**为什么？**
- 直接说喜欢 = 需求感过强 = 价值降低
- 模糊表达 = 保持神秘 = 对方去猜

### 3. 没睡过，不能说"我爱你"

**原则：** 身体关系没到那一步，话不能说到那一步

**错误：**
- 聊了几天就说"我爱你"
- 还没见面就深情表白

**正确：**
- 用行动表达，不用嘴说
- 等睡过之后再说
- 或者永远不说，用行动证明

### 4. 表达好感要含蓄

| 直接表达（错误） | 含蓄表达（正确） |
|---|---|
| 我喜欢你 | 和你聊天挺开心的 |
| 我好想你 | 最近有空吗，出来坐坐 |
| 你是我最特别的人 | 你和别人不太一样 |
| 我想见你 | 在哪呢，顺便见个面 |

### 5. 承诺要少，行动要多

**不要说：**
- "我会对你好的"
- "我会一直陪着你"
- "我永远不会离开你"

**要做的：**
- 真的对她好
- 真的陪着她
- 用行动证明

## 话术分级

### 等级1：刚认识
**可以说：**
- "你是做什么的？"
- "平时喜欢干什么？"
- "你人挺有意思的"

**不能说：**
- 任何暧昧的话
- 任何承诺
- 过度夸赞

### 等级2：聊得来
**可以说：**
- "你挺好的"
- "我觉得你也挺不错的"
- "和你聊天挺开心"
- "改天出来坐坐"

**不能说：**
- "我喜欢你"
- "我想和你在一起"
- "我爱你"

### 等级3：暧昧期
**可以说：**
- "你和别人不一样"
- "越来越喜欢和你聊天了"
- "想你了"
- "你在干嘛"

**不能说：**
- "我爱你"（除非睡过）
- 过度承诺
- 太直接的表达

### 等级4：睡过之后
**可以说：**
- 慢慢表达真实情感
- 适度表达喜欢
- 但依然不要说太满

**原则：** 即使睡过，话也不要说太满

### 等级5：确定关系
**可以说：**
- 正常表达情感
- "我爱你"
- 承诺和未来

## 实战原则

### 1. 永远比对方少说一点
- 对方说10分，你说7分
- 保持你的价值感
- 不让关系太快

### 2. 用行动代替说话
- 想她 = 找她出来
- 喜欢她 = 对她好
- 爱她 = 用行动证明

### 3. 话别说太满，留有余地
- 不要说"永远"
- 不要说"一定"
- 不要说"保证"

### 4. 关系没到，话不要说
- 没睡过 = 不说"我爱你"
- 没确定 = 不说"你是我的"
- 没深度 = 不说深度话题

---

## 常见错误

### 错误1：太快表白
- 刚认识几天就表白
- 还没见面就说喜欢
- **后果：** 需求感过强，把对方吓跑

### 错误2：话说太满
- "我会永远对你好"
- "我永远不会离开你"
- **后果：** 压力太大，对方想逃

### 错误3：直接说喜欢
- "我喜欢你"
- "我好爱你"
- **后果：** 价值感降低，失去神秘感

### 错误4：关系没到话说到
- 还没睡过就说"我爱你"
- 还没确定就说"你是我的"
- **后果：** 节奏不对，关系崩盘
