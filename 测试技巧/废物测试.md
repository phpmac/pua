# 废物测试（Shit Test）

## 什么是废物测试？

**定义：** 对方用来测试你的价值、底线、自信的各种问题或行为。

**本质：** 女性在评估你是否值得她投入，测试你的"价值"和"框架"。

---

## 我们要反过来用

**主动抛出废物测试**，判断对方的反应和类型。

---

## 常见废物测试

### 1. 价值测试

**目的：** 测试你的自信程度

**话术示例：**
- "你看上去不像XXX的人"
- "你好像不太合群"
- "你看上去年龄挺大的"

**对方反应判断：**
- 急于解释 = 价值低，不自信
- 淡定回应/幽默化解 = 价值正常
- 反过来调侃 = 价值高

### 2. 底线测试

**目的：** 测试你的底线在哪里

**话术示例：**
- "帮个忙呗"
- "请我吃个饭"
- "借我点钱"

**对方反应判断：**
- 立刻答应 = 没底线，容易被拿捏
- 婉拒但愧疚 = 底线不够坚定
- 淡定拒绝 = 有底线
- 反客为主 = 掌控力强

### 3. 投入测试

**目的：** 测试你愿意为她投入多少

**话术示例：**
- "你能来接我吗？"
- "能不能帮我XXX"
- "我好难过"（看你会不会花时间）

**对方反应判断：**
- 过度热情 = 投入过度，容易被操控
- 适度回应 = 正常
- 保持框架 = 掌控节奏

### 4. 情感测试

**目的：** 测试你的情感状态

**话术示例：**
- "你有女朋友吗？"
- "你前女友怎么样？"
- "你喜欢什么样的？"

**对方反应判断：**
- 过度透露 = 信息泄露过多
- 模糊回答 = 有经验
- 反问 = 高手

### 5. 暧昧测试

**目的：** 测试你对她的兴趣程度

**话术示例：**
- "你觉得我怎么样？"
- "你会喜欢我吗？"
- 各种暗示性话语

**对方反应判断：**
- 立刻表白 = 需求感过强，被动
- 保持神秘 = 有经验
- 反过来撩 = 高手

---

## 如何应对对方的废物测试

### 核心原则
1. **不解释、不辩解** - 解释就是输了
2. **保持框架** - 不被对方带节奏
3. **幽默化解** - 化解压力，保持轻松
4. **反客为主** - 把测试抛回给对方

### 具体技巧

#### 幽默化解
**对方：** "你看上去不像有钱人"
**回答：** "所以需要你来扶贫啊"

#### 反客为主
**对方：** "你有女朋友吗？"
**回答：** "你打算应聘吗？"

#### 淡定无视
**对方：** "你好像不咋地"
**回答：** "哦"（继续说自己的）

#### 模糊回应
**对方：** "你觉得我怎么样？"
**回答：** "还行吧"

## 实战应用流程

### 第一步：抛出测试
选择合适的测试话术，观察对方反应

### 第二步：分析反应
根据对方的回应，判断她的类型和价值

### 第三步：调整策略
- 价值低的：可以继续
- 价值高的：需要更多技巧
- 套路深的：果断放弃

### 第四步：保持框架
无论对方什么反应，你都要保持自己的节奏

## 测试话术库

### 初始阶段
- "你看上去像是XXX类型的人"
- "你好像不太会聊天"
- "你看起来年龄好小"

### 了解阶段
- "你前男友一定很XXX吧"
- "你是不是经常这样"
- "我感觉你有点XXX"

### 深入阶段
- "你是不是对我有意思"
- "你好像变了"
- "我们算什么关系"
