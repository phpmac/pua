## 快速开始

### 1. 识别对方是不是套路
- 查阅 `识别技巧/套路识别.md`
- 数"钱"字出现次数判断酒托套路

### 2. 主动测试对方
- 参考 `测试技巧/废物测试.md`
- 抛出测试，观察反应

### 3. 控制交流节奏
- 学习 `拿捏技巧/节奏控制.md`
- 掌握回复时间、主动权、情感投入

### 4. 高情商回复
- 使用 `聊天技巧/回复模板.md`
- "相信一见钟情" → "我会相信的"

## 核心经验法则

### 金钱套路判断
| "钱"字出现次数 | 判断 |
|---|---|
| 3次以上 | 高度警惕 |
| 5次以上 | 基本确定套路 |
| 10次以上 | 100%酒托 |

### 废物测试原则
- 不解释、不辩解
- 幽默化解
- 反客为主
- 保持框架

### 节奏控制要点
- 对方快你慢，对方慢你稳
- 回复不要太快
- 热情不要过度
- 你是节奏的主导者
