# 夜场套路模式（成都骗术）

## 什么是"骗术"

夜场/夜店圈子里，**"骗术"** = 指经过培训的套路剧本。

目的：吊住客户，让客户持续投入金钱，但不实际交付。

---

## 常见套路模式

### 模式1：拖延吊凯子型

**目标：** 保持线上联系，但不实际见面，持续吊着

**剧本流程：**
```
1. 答应见面（建立期待）
2. 见面前说"没钱"（测试你是否会主动给钱）
3. 找理由爽约（测试底线）
4. 再次约但继续拖延（保持期待）
5. 标准拖延话术：
   - "在收拾东西"
   - "在洗头"
   - "在洗澡"
   - "收拾家里"
6. 你发真实位置不敢来（只敢线上）
7. "以为你把我删了"（回吸话术）
```

**识别方法：**
- 每次见面必定有借口
- 借口都是琐事（洗头洗澡最常见）
- 发酒店/具体位置不敢来
- 线上热情，线下冷淡

**真实目的：**
- 吊着你作为备胎/潜在客户
- 在KTV有客人时优先陪客人
- 没客人才想找你打发时间/要钱

---

### 模式2：酒托型

**目标：** 让你买单高额消费

**剧本流程：**
```
1. 约你去特定酒吧/餐厅
2. 点超贵的酒水
3. 中途借故离开（上厕所/接电话）
4. 你买单后她消失
5. 或者让你"先付钱，我马上来"
```

**识别方法：**
- 指定去某个地方（不是你选的）
- 点单时很熟练地点贵的
- 从来不提AA
- 对话中"钱"字出现次数多

---

### 模式3：借钱周转型

**目标：** 借钱不还

**剧本流程：**
```
1. 建立一段时间的感情
2. 突然说"急用钱"
3. 理由：
   - 房租到期
   - 家人生病
   - 手机坏了
   - 被朋友坑了
4. 承诺"过几天还你"
5. 钱到手后冷淡/消失
```

**识别方法：**
- 刚认识不久就借钱
- 理由听起来很紧急
- 承诺还钱但从不主动提
- 一提到还钱就各种借口

---

### 模式4：感情绑架型

**目标：** 用感情让你持续投入

**剧本流程：**
```
1. 快速升温感情
2. 让你产生"她是真的喜欢我"的错觉
3. 开始要求：
   - "你都不愿意为我花钱"
   - "你是不是不爱我"
   - "别人男朋友都XXX"
4. 你一犹豫就冷战/消失
5. 你妥协了再出现
6. 循环往复
```

**识别方法：**
- 升温太快不正常
- 总是拿"爱不爱"说事
- 一提钱就热情，不给钱就冷淡
- 用内疚感控制你

---

## 标准话术库

### 拖延话术
- "在收拾东西，马上就好"（永远不会好）
- "在洗头，洗完就来"（洗不完的头）
- "在洗澡，一会说"（最经典，洗澡不能回消息）
- "收拾家里，乱死了"（家里永远收拾不完）

### 爽约话术
- "陪客人去了，没办法"
- "临时有事"
- "被朋友叫走了"
- "睡过头了"

### 回吸话术
- "以为你把我删了"
- "怎么不理我了"
- "在吗"
- "最近忙吗"
- "突然想起你了"

### 要钱话术
- "没钱了，帮个忙"
- "套现手机ID"
- "急用钱，过几天还"
- "你都不愿意为我花钱"
- "别人男朋友都XXX"

## 培训重点（她们被教的内容）

### 核心原则
1. **不要撕破脸** - 客户永远是潜在的
2. **保持联系** - 不要彻底断掉
3. **不拒绝但拖延** - 不直接说no，但永远不兑现
4. **装傻装无辜** - 被揭穿就装不懂
5. **回吸话术** - 消失一段时间再出现

### 应对客户揭穿
**客户：** "你这套路太low了"
**她的反应：**
- 装不懂："什么套路？"
- 转移话题："你怎么这么想"
- 倒打一耙："你根本不相信我"
- 卖惨："我也是没办法"

### 情绪控制
- 被骂不生气（保持客户）
- 被揭穿不承认（装傻）
- 客户冷淡时主动找（回吸）
- 客户热情时保持距离（吊着）

## 判断标准

### 高度疑似有剧本
- [ ] 话术和标准套路高度相似
- [ ] 节奏太精准（每次都在关键时刻出问题）
- [ ] 不撕破脸，装傻装无辜
- [ ] 使用回吸话术
- [ ] 线上热情线下冷淡

### 100%确定
- [ ] 以上特征全中
- [ ] 你揭穿后她继续用套路话术
- [ ] 发真实位置不敢来
- [ ] 提到钱就很积极

## 应对策略

### 最优解：直接删除拉黑
- 这种人投入产出比为负
- 时间成本远高于可能的收获
- 她们有无数个"客户"

### 如果非要测试

**终极测试话术：**
```
"既然你一直说没钱，那我帮你。你现在过来，我给你2000块现金。"
```

**判断：**
- 立刻答应 = 100%酒托/托儿
- 继续找借口 = 没诚意，删
- 愤怒/骂你 = 被戳中痛处

**但说实话，不用测了，答案已经很明确。**

## 实战案例记录

### 案例1：成都KTV女生（路遥案例）

**情况：**
- KTV认识，约好两次吃饭
- 第一次：爽约说陪客人
- 第二次：说没钱要套现手机ID
- 各种拖延：收拾东西、洗头洗澡
- 揭穿她装不懂
- 发酒店位置不敢来，发打桌球也不来
- 现在说"以为你把我删了"

**套路类型：** 拖延吊凯子型

**判断：** 有人教的套路，剧本痕迹明显

**建议：** 直接删除拉黑

**经验总结：**
- 夜场女生的"没钱"是常态，但每次见面都说就是套路
- "洗头洗澡"是经典拖延话术
- 发真实位置不敢来 = 只想吊你线上
- "以为删了"是标准回吸话术
