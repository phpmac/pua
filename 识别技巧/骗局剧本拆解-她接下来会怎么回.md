# 骗局剧本拆解：她接下来会怎么回

## 背景

你发：`没删，怎么了？`

**她的心理状态：**
- 她发现你没有删她（或者重新加回来了）
- 她在测试你是否还在她的控制范围内
- 她想重新建立连接，继续吊着你

---

## 她可能的回复类型

### 类型1：装傻装无辜型（概率：40%）

**她的回复：**
```
"没事啊，就是看你看不到朋友圈，以为你把我删了。"
```
```
"没有啦，就是看你看不到我朋友圈，以为你把我删了嘛。"
```
```
"就随便问问嘛，怎么了？"
```

**她的套路：**
- 装作无所谓，"就是随便问问"
- 把问题转移成"她担心你删了她"
- 装无辜，好像真的只是误会

**她的真实目的：**
- 测试你是否还在意她
- 重新建立联系，但不暴露她的需求感
- 保持高位，让你觉得是她"担心"被删，而不是她需要你

**你的应对：**
- 不要解释为什么屏蔽朋友圈
- 不要说"我没删你啊"这种解释的话
- 保持淡定，可以不回或者简短回复

## 骗局核心模式分析

### 夜场女生的套路流程

```
第1步：建立联系（已完成）
    ↓
第2步：制造期待（"请吃饭"）
    ↓
第3步：测试底线（爽约看你是否容忍）
    ↓
第4步：金钱测试（"没钱"看你是否给钱）
    ↓
第5步：保持吊着（各种拖延，不实际见面）
    ↓
第6步：回吸话术（"以为删了"重新建立连接）
    ↓
第7步：循环重复（回到第2步）
```

### 她的核心目标

| 目标 | 手段 | 识别方法 |
|------|------|---------|
| 让你花钱 | "没钱"、"套现" | 每次见面前提钱 |
| 让你容忍 | 爽约、拖延 | 看你是否生气 |
| 让你等待 | 收拾东西、洗头 | 永远在准备中 |
| 让你投入 | 情感绑架 | "你不爱我了吗" |
| 保持吊着 | 线上热情线下冷 | 发真实位置不敢来 |

## 如何防范这种骗局

### 防范法则

**法则1：不见面不给钱**
- 线上聊得再好也没用
- 真人见面之前不要有任何金钱往来
- "没钱"是她的事，不是你的事

**法则2：爽约一次就降低期望**
- 第一次爽约可能是意外
- 第二次爽约就是故意的
- 不要相信第三次

**法则3：发真实位置测试**
- 约酒店/具体地方
- 不敢来 = 只想吊你线上
- 敢来 = 还有诚意（但也要警惕）

**法则4：不解释不辩解**
- 她问你问题，你反问回去
- 不要解释你的行为
- 不要为你的决定辩解

**法则5：保持框架**
- 你是高位，她是低位
- 你是选择者，她是被选择者
- 你随时可以离开，她不可以

## 识别关键词

**如果她经常说这些话，警惕：**

| 话术 | 说明 |
|------|------|
| "没钱" | 金钱测试，看你会不会主动给 |
| "套现" | 继续金钱测试，而且说得很具体 |
| "陪客人" | 爽约借口，也是她真实工作 |
| "在洗头/洗澡" | 经典拖延话术 |
| "收拾东西" | 拖延话术 |
| "以为删了" | 回吸话术，重新建立连接 |
| "你都不爱我了" | 情感绑架 |
| "别人男朋友都XXX" | 情感绑架/比较 |
